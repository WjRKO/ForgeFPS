# Mini-guide per il server Discord FrameForge

Contiene i testi pronti da incollare come **messaggi pinnati** nei canali principali del server.
Puoi copiare esattamente il markdown Discord (grassetto, link, emoji) e Discord lo renderizza automaticamente.

Ricorda: **prima di postare**, sostituisci `#nome-canale` e `@Nome-Ruolo` con i nomi reali del tuo server.

---

## 📌 Pin in `#benvenuto` (o `#regole`)

```
👋 **Benvenuto in FrameForge!**

Siamo la community di gamer e streamer che vogliono spremere ogni FPS dal proprio PC — senza dover fare tweak a mano.

**Come iniziare in 30 secondi:**
1. 📖 Fai `/come-iniziare` qui su Discord (o vai su https://forgefps.dev/guida)
2. 🔗 Fai `/link` per collegare il tuo account FrameForge (otterrai il ruolo **Boosted PC**)
3. 💬 Presentati in <#PRESENTAZIONI> e dicci PC + giochi preferiti

**Cosa puoi fare qui:**
• `/mypc` — mostra il tuo Health Score
• `/benchmark` — l'ultimo punteggio
• `/leaderboard` — top 10 utenti
• `/ruoli` — lista dei ruoli e come ottenerli
• `/canali` — mappa dei canali
• `/help` — tutti i comandi

**Sei streamer o creator?**
Fai `/apply-creator link:<tuo canale Twitch/YouTube/Kick>` per candidarti al ruolo **Creator Verified**.

Buon boost! 🚀
```

## 📌 Pin in `#regole`

```
📜 **Regole del server**

1. **Rispetto** — Niente insulti, discriminazioni, hate speech. Punto.
2. **No spam** — Un messaggio ripetuto o autopromozione senza contesto = warn.
3. **No cheat / crack** — Niente link a hack, crack, key generator. Ban immediato.
4. **Off-topic nel canale giusto** — Meme e chat libera in <#CHAT-GENERALE>, non altrove.
5. **Autopromozione** — Il tuo canale Twitch/YouTube va in <#SHOWCASE>, non nella chat generale. Chiedi il ruolo Creator Verified con `/apply-creator`.
6. **DM ai membri** — Vietato spammare in DM. Se qualcuno lo fa a te, segnalalo allo <@&STAFF>.
7. **Discussioni tecniche** — Domande su tweak, hardware, driver? Vai in <#AIUTO>.
8. **Lingua** — Italiano ed English sono benvenuti. Sii chiaro con chi ti risponde.

**Sanzioni:**
- 1° warn → richiamo scritto
- 2° warn → mute 24h
- 3° warn → ban temporaneo (7 giorni)
- Regola grave (cheat, insulti pesanti) → ban immediato

Grazie di rendere questo posto migliore per tutti. 💛
— Il team FrameForge
```

## 📌 Pin in `#annunci` o in cima a `#showcase`

```
🎬 **Vuoi il ruolo Creator Verified?**

Il ruolo **Creator Verified** è riservato a streamer e content creator che pubblicano regolarmente contenuti gaming/hardware/streaming.

**Come candidarti:**
1. Fai `/apply-creator link:https://twitch.tv/tuocanale` (funziona anche con YouTube e Kick)
2. Lo staff riceve una notifica nel canale privato e revisiona il tuo canale
3. Ricevi un DM con l'esito entro qualche giorno

**Requisiti indicativi** (non tassativi):
• Canale attivo negli ultimi 3 mesi
• Contenuti pertinenti (gaming, hardware, streaming, benchmark, tech)
• Nessuna violazione delle policy Discord/piattaforma

**Vantaggi del ruolo:**
• Nome colorato + emoji dedicato
• Accesso a canali riservati Creator (in arrivo)
• Menzioni nelle release e nei showcase social

Buona fortuna! 🎥
```

## 📌 Pin in `#aiuto` (o `#faq`)

```
🆘 **Come chiedere aiuto qui**

Prima di scrivere, aiutaci ad aiutarti:

**Fornisci queste info:**
1. 🖥️ **Hardware**: CPU, GPU, RAM, Windows version (usa `/mypc` per copiarli)
2. 🎮 **Cosa stai facendo**: gioco/app + risoluzione + preset
3. ❌ **Cosa non funziona**: FPS, crash, latenza, driver, altro
4. 📸 **Screenshot** dell'errore o del comportamento anomalo, se disponibile
5. 🧪 **Cosa hai già provato** (evita risposte tipo "prova a reinstallare i driver" se lo hai già fatto)

**Risorse rapide:**
• [Guida completa](https://forgefps.dev/guida)
• [Changelog](https://forgefps.dev/changelog) — bug noti + fix pubblicati
• `/mypc` — screenshot del tuo Health Score
• [Segnala un bug su GitHub](https://github.com/WjRKO/ForgeFPS/issues/new/choose)

Non sei mai da solo: la community risponde 💛
```

## 🎨 Suggerimenti extra

- **Server Discovery**: attiva la funzione *Discovery* dalle impostazioni server per far apparire il server nelle ricerche.
- **Server Insights**: monitora crescita e attività da *Impostazioni server → Insights*.
- **Auto-mod**: attiva la moderazione automatica per parole vietate e link sospetti (*Impostazioni server → AutoMod*).
- **Onboarding integrato Discord**: *Impostazioni server → Onboarding* per creare un mini-tour nativo (compatibile con i nostri slash commands).
